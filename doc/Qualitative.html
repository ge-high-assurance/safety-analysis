module Qualitative: sig .. end
The top-level functions for the qualitative manipulation of fault trees are as follows.

  • cutsets : a function for generating the cutsets of a fault tree.
FORMULAS

Formulas used in modeling are represented with the formula and cformula types. The top-level functions for manipulating formulas are as follows. Typical users can ignore these functions.

  • formulaOfTree : a function for translating a tree into a formula.
  • ssfc : a function for simplifying formulas.
  • sop : a function for generating the simplest DNF formula equivalent to its input.

val ssfc : 'a FaultTree.pexp -> 'a FaultTree.pexp
This is a function that given a formula simplifies it by applying constant propagation, flattening, removal of duplicate literals and subexpressions, and also sorts the resulting formula so that simpler terms are shown first. This is done until a fixpoint is reached.

val sop : 'a FaultTree.pexp -> 'a FaultTree.pexp
Given any monotone Boolean formula, sop returns a simplified formula in DNF (sum of products form). The expectation is that it returns the simplest equivalent DNF formula.
