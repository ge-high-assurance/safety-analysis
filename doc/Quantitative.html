Module Quantitative

module Quantitative: sig .. end
The top-level functions for the quantitative analysis of fault trees are as follows.

  • probErrorCut : a function that given a fault tree, computes the top-level probability of a fault.
  • probErrorCutImp : a function that given a fault tree, computes importance measures.
If cutsets are available, the next two functions can compute probabilities and importance metrics without having to recompute them.

  • probErrorCutC : a function that given a fault tree and cutsets, computes the top-level probability of a fault.
  • probErrorCutCImp : a function that given a fault tree and cutsets, computes importance measures.
