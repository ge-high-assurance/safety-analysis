FAULT TREES

Fault trees are represented with the ftree type and monotone Boolean formulas are represented with the pexp type.

type 'a ftree =
| Leaf of 'a * float * float
| SUM of 'a ftree list
| PRO of 'a ftree list
A fault tree is either: a leaf containing a failure rate and exposure time OR an (n-ary) operator along with a list of trees. *
type 'a pexp =
| TRUE
| FALSE
| Var of 'a
| Sum of 'a pexp list
| Pro of 'a pexp list
Type definitions for monotone Boolean formulas.
